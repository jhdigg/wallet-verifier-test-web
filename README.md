## Run with Docker

```bash
npm install
npm run generate
docker compose  -f ./docker/docker-compose.yml up -d
```

Then open [http://localhost](http://localhost) in your browser.

## Live Development

After starting Docker (see above), run:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.
